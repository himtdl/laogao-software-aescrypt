# Changelog

本项目所有显著变更均会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [1.1.2] - 2026-04-25

### 新增
- 窗口标题改为 `[老高之家]AES 加解密小工具 - 禁止用于非法用途 | 后果自负`
- 勾选“删除源文件”时弹出安全警告对话框
- 密码内存安全处理：引入 `zeroize`，使用 `Zeroizing<String>` + `Drop` 确保密码残留被清除

### 变更
- **移除“记住密码”功能**及相关依赖 (`base64`, `dirs`)，规范密码处理流程
- 文件对话框初始目录改为 EXE 所在目录
- 加密/解密统一要求 **必须输入密码** 才可点击开始
- **加密失败后保留选中文件**，路径与密码不清除，方便重试
- 窗口支持自由拉伸，并设定最小尺寸 480x400

### 修复
- 强化 `Cargo.toml` 编译配置：`strip = symbols`、`panic = abort`、`lto = true` 等
- 使用 `obfstr!` 宏混淆所有用户可见字符串

## [1.1.0] - 2026-04-24

### 新增
- 输出目录自定义选择，默认仍为程序所在目录
- 加密功能：支持任意文件加密成 `.aes` 格式
- 加密环境选项：输出目录选择、加密后删除源文件、文件名长度检查（250 字符限制）
- 解密失败不清除文件选择，保留路径和密码

### 变更
- 版本号从 1.0.2 升级至 1.1.0

## [1.0.2] - 2026-04-24

### 变更
- 改为纯 GUI 模式 (`#![windows_subsystem = "windows"]`)，屏蔽 CMD 窗口
- 窗口标题修改为工具名称

## [1.0.0] - 2026-04-24

### 新增
- 自动检测并加载系统中文字体（微软雅黑等），避免乱码
- 解码超过 30 秒时显示“取消”按钮

### 变更
- UI 重构，状态管理更清晰

## [0.1.0] - 2026-04-24

### 新增
- 首个发布版本
- 支持 AES 文件解密（`.aes`）
- 防多开机制
- 内存安全密码处理
- 基础 GUI 界面
